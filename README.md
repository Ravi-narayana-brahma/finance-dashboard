# 💰 Finance Dashboard UI

A clean and interactive **Finance Dashboard** built to visualize and manage financial data in a simple and user-friendly way.

This project was developed as part of a frontend assignment to demonstrate UI design, component structuring, and state management skills.

---

## 🚀 Live Demo

👉 https://finance-dashboard-rnb.netlify.app/

## 📂 GitHub Repository

👉 https://github.com/Ravi-narayana-brahma/finance-dashboard.git

---

## 📌 Objective

The goal of this project is to build a **frontend-only financial dashboard** that allows users to:

* View financial summaries
* Explore transactions
* Understand spending patterns
* Interact with a clean and responsive UI

---

## 🧠 Features Implemented

### 📊 Dashboard Overview

* Summary cards:

  * Total Balance
  * Income
  * Expenses
* Time-based visualization (Balance trend)
* Category-based visualization (Spending breakdown)

---

### 💳 Transactions Section

* Transaction list with:

  * Date
  * Amount
  * Category
  * Type (Income / Expense)
* Features:

  * 🔍 Search functionality
  * 🔄 Filtering
  * 📅 Sorting

---

### 👤 Role-Based UI (Frontend Simulation)

* **Viewer**

  * Can only view data
* **Admin**

  * Can add and manage transactions
* Role switch implemented using UI toggle

---

### 📈 Insights Section

* Highest spending category
* Monthly comparison
* Key financial observations

---

### ⚙️ State Management

* Managed using React Hooks:

  * `useState`
  * `useEffect`
* Handles:

  * Transactions
  * Filters
  * Role selection

---

### 📱 Responsive Design

* Fully responsive across:

  * Mobile 📱
  * Tablet 💻
  * Desktop 🖥️

---

## 🎨 UI/UX Highlights

* Clean and modern design
* Smooth hover and transition effects
* Dark mode support 🌙
* Handles empty states gracefully

---

## 🛠️ Tech Stack

* **Frontend:** React.js (JavaScript)
* **Styling:** Custom CSS
* **Charts:** Recharts / Chart.js
* **State Management:** React Hooks

---

## ⚖️ Technical Decisions & Trade-offs

* Used **React.js** for modular and reusable components
* Chose **custom CSS** instead of frameworks for full design control
* Used **local state (useState)** instead of Redux for simplicity
* Used chart libraries to simplify data visualization
* Focused on **clean UI and usability over complexity**

## 📦 Setup Instructions

1. Clone the repository

```bash
git clone https://github.com/your-username/finance-dashboard.git
```

2. Navigate to project folder

```bash
cd finance-dashboard
```

3. Install dependencies

```bash
npm install
```

4. Run the project

```bash
npm start
```
